# 💡 פרויקט סיום Patenta –  

## 📋 סקירת המיזם
**Patenta** הוא אתר קונספט המדמה חברת סטארט-אפ טכנולוגית. המיזם מאפשר ליזמים להפוך רעיון מופשט למוצר פיזי בשוק. האתר משלב חנות למוצרי פטנט קיימים יחד עם מערכת הצטרפות (Onboarding) ליזמים חדשים.

---

## 🗺️ מפת דרכים למבנה הפרויקט (Project Roadmap)

הפרויקט בנוי בצורה מודולרית המפרידה בין התוכן לעיצוב, כדי להבטיח קוד קריא וקל לתחזוקה:

### 1. דפי המערכת (Core Pages)
הפרויקט כולל 5 דפים עיקריים המכסים את כל מסע המשתמש:
* **`index.html`** - דף הנחיתה הראשי. מציג את חזון החברה, המומחים שלנו ואזור "אודות".
* **`Proudct.html`** - קטלוג המוצרים המלא הכולל כרטיסיות מוצר מעוצבות.
* **`des_proudct.html`** - דף המפרט את תהליך הפיתוח (Design Process).
* **`form.html`** - טופס שליחת רעיון והצטרפות לקהילה.
* **`thanks.html`** - דף אישור (Confirmation) לאחר שליחת טופס.

### 2. ארכיטקטורת עיצוב (Styling Architecture)
העיצוב מפוצל ל-7 קבצי CSS כדי לשמור על סדר:
* **`cssAll.css`** - הגדרות גלובליות, פונטים, Header ו-Footer (משותף לכל הדפים).
* **`home.css`** - עיצוב ייחודי לדף הבית הכולל את האנימציות הראשיות.
* **`proudct.css`** - עיצוב הגריד והכרטיסיות של הקטלוג.
* **קבצי עיצוב ייעודיים** נוספים לשאר דפי התוכן.

## 🛠️ דגשים טכניים וביצועים

### חלוקה סמנטית (Semantic HTML)
בניתי את האתר בחלוקה קשיחה לתגיות סמנטיות:
* **`<header>`**: תפריט ניווט קבוע עם אפקטים של Hover.
* **`<main>`**: התוכן הדינמי המשתנה בין דף לדף.
* **`<footer>`**: אזור יצירת קשר וניתוב מהיר (Quick Links).



### מערכת Flexbox
כל סידור האלמנטים באתר מבוסס על מודל ה-Flex. זה אפשר לי לשלוט בצורה מדויקת ביישור (Alignment) ובפיזור המרכיבים בתוך הדיבים, במיוחד בקטעים מורכבים כמו כרטיסיות המוצרים וה-Nav.

### חווית למידה: "מהקוד למציאות"
האתגר הגדול ביותר עבורי היה המעבר מעיצוב ויזואלי לכתיבת קוד. בחרתי לעבוד ידנית עם פקודות ה-Flexbox ולא בתוכנות גרירה-והשלכה (Drag & Drop), מתוך הבנה שזה מה שיחשל אותי כמתכנתת. התהליך הזה הטמיע בי את ההבנה שכל תגית וכל שורת קוד משפיעה על חווית המשתמש הסופית.
